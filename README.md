# Hero Dev Portfolio ✨

## Description
This repository features a clean, single-page portfolio designed to showcase a frontend developer's profile, core technical skills, and recent work history. Built as a practical application of foundational web layout concepts, this project translates a static Figma design into structured web elements entirely from scratch using vanilla HTML and CSS.

## Preview
A static portfolio-style landing page featuring:
- **Hero Section:** Introduction with call-to-action buttons.
- **Skills Showcase:** A visual grid highlighting core web technologies.
- **Projects Section:** Alternating cards displaying featured development works.
- **Experience Section:** A layout grid showcasing trusted brands.
- **Footer:** Copyright information and integrated social media links.

## Built With
- HTML5
- CSS3 (Flexbox)
- Google Fonts (*Plus Jakarta Sans*)
